## Prerequisites

Before performing an ATM transaction, ensure the following:

- Valid Debit Card
- Personal Identification Number (PIN)
- Sufficient account balance

---

## Cash Withdrawal Procedure

1. Insert the Debit Card into the ATM.
2. Select **Withdraw Cash**.
3. Select **Savings Account**.
4. Enter the withdrawal amount.
5. Verify withdrawal limits.
6. Enter Debit Card PIN.
7. Collect cash.
8. Collect transaction receipt.
9. Remove Debit Card.
10. ATM closes the transaction.

## Appendix

### Transaction Limits

| Transaction Type | Maximum Limit |
|------------------|---------------|
| Cash Withdraw | ₹20,000 per transaction |
| Balance Enquiry | No limit |
| Mini Statement | No limit |

### Common Error Messages

| Error Message | Description |
|----------------------|------------------------------|
| Invalid PIN | Incorrect PIN entered |
| Insufficient Balance | Account has insufficient funds |
| Transaction Declined | Bank rejected transaction |

### Safety Instructions

- Do not share ATM PIN.
- Cover keypad while entering PIN.
- Collect card and receipt after transaction.
- Use ATMs in secure locations.
- Contact bank if cash is not dispensed.
